# Weight Validation Tests

This project demonstrates automated Python test cases for validating weight-based logic
using minimum and maximum thresholds.

## Purpose
- Simulate industrial-style weight validation logic
- Demonstrate test automation and edge case handling
- Apply QA testing principles in Python

## Project Structure
src/
├── validation.py      # Core validation logic
└── tests/
    └── test_validation.py  # Automated test cases

## Covered Scenarios
- Weight below minimum threshold
- Weight above maximum threshold
- Weight within valid range
- Invalid input handling

## Technologies
- Python
- Pytest-style assertions
